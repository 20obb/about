# 0xdevil Portfolio

A modern, responsive personal portfolio website showcasing skills, projects, and contact information.

## Features

- **Responsive Design** - Fully responsive layout that works on all devices
- **Dark/Light Theme** - Toggle between dark and light modes
- **Smooth Animations** - Engaging animations and transitions
- **Typing Effect** - Dynamic typing animation in the hero section
- **Interactive Navigation** - Smooth scrolling and mobile-friendly hamburger menu

## Sections

- **Home** - Hero section with introduction and social links
- **About** - Personal information, experience, and background
- **Skills** - Technical skills and proficiencies
- **Projects** - Portfolio of completed work
- **Contact** - Contact form and social media links

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts (Inter, Fira Code)

## Setup

1. Clone the repository
2. Open `index.html` in your browser

## File Structure

